#  Java Practice Project (Tab 2)

📌 Overview

This project is a collection of **Java programs** organized by topics to help improve:

- Programming logic
- Problem-solving skills
- Core Java concepts
- Object-Oriented Programming (OOP)

The code is written in a **simple and structured way**, making it easy to understand for beginners.

---

 📂 Project Structure

Tab_2/

├── Numbers/  
├── Array_Programs/  
├── String_Programs/  
├── Collections_Programs/  
├── OOP_Concepts/  

---

 1. Number Programs

📁 Folder: `Numbers/`

Programs included:
- Swap two numbers without using a third variable  
- Find missing number  

✔ Focus: Basic logic and mathematical operations  

---

 2. Array Programs

📁 Folder: `Array_Programs/`

Programs included:
- Find second largest number in array  
- Rotate array by k positions  
- Find missing number  
- Remove duplicates (List → Set → List)  

✔ Focus: Array manipulation and logic building  

---

 3. String Programs

📁 Folder: `String_Programs/`

Programs included:
- Check if two strings are anagrams  
- Find longest word in a sentence  
- Remove all spaces from a string  
- Check balanced parentheses  
- Count words in a sentence  
- Print duplicate characters  

✔ Focus: String handling and pattern logic  

---

 4. Collections Programs

📁 Folder: `Collections_Programs/`

Programs included:
- Find top 3 highest marks from student list  
- Remove duplicates using HashSet  
- Sort employees by salary (descending)  
- Group students by marks using HashMap  

✔ Focus: Java Collections Framework (List, Set, Map)  

---

 5. OOP Concepts

📁 Folder: `OOP_Concepts/`

 🔹 Library Management System
- Classes: Book, Member  
- Features: Issue and return books  

 🔹 Inheritance (Vehicle System)
- Parent: Vehicle  
- Child: Car, Bike  

 🔹 Abstraction (Shape)
- Abstract class: Shape  
- Child classes: Circle, Rectangle  

 🔹 Interface (Payment System)
- Interface: PaymentMethod  
- Implementations: CreditCard, UPI  

✔ Focus: Real-world OOP design  

---

 🚀 How to Run

1. Open the project in any IDE:
   - IntelliJ IDEA  
   - Eclipse  
   - VS Code  

2. Compile the program:
   javac FileName.java  

3. Run the program:
   java FileName
