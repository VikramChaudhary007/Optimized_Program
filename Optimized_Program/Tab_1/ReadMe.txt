📘 Java Programming Tasks – Basic to Mini Project
📌 Overview

This repository contains a collection of Java programs covering:

Basic programming concepts

Core Java fundamentals

Logical problem-solving

Mini project (CRUD application)

These tasks are designed to build a strong foundation in Java programming and problem-solving skills.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

🧩 1. Basic Programming (Easy Level)

⏱️ Time: 30–45 minutes

🔢 Task 1 – Number Operations
📋 Description

Write a Java program to:

Take a number as input

Check whether the number is:

Even or Odd

Prime or Not

💡 Concepts Used

Conditional statements (if-else)

Loops

Mathematical logic

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

🔤 Task 2 – String Handling
📋 Description

Write a Java program to:

Take a string as input

Perform the following operations:

Reverse the string

Count number of vowels

Find string length

💡 Concepts Used

String handling

Loops

StringBuilder (optimized reverse)
